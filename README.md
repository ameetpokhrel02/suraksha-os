# 🛡️ Suraksha OS | सुरक्ष्या OS

> **Empowering Nepali Citizens Against Cyber Threats through Interactive Education.**

---

## ⚡ Safety & Education First (Not for Hacking)

> [!IMPORTANT]
> **Suraksha OS is purely an educational awareness simulator.** 
> It is designed to train non-technical citizens, students, and internet users on how to recognize cyber attacks, identify phishing indicators, and safeguard their digital identity. It **does not contain or promote any offensive security, hacking tools, or malware**.

---

## 🛠️ The Tech Stack

| Technology | Role in System | Details / Highlights |
| :--- | :--- | :--- |
| **🌐 HTML5** | Structural Layout | Semantic markup, localized for high-readability and standard search optimization. |
| **🎨 Vanilla CSS3** | Custom UI Styling | Sleek dark-mode grid layouts, neon glow highlights, glassmorphic widgets, and responsive screen flow. |
| **⚡ Vanilla JavaScript** | Interactive Engine | Zero external dependencies. Drives the terminal interface, phishing simulators, password checker, and local state management. |

---

## 📱 Interactive View Toggle (NEW)
Suraksha OS features a **Dynamic Preview Mode** allowing users to switch between a **🖥️ Desktop Layout** and a **📱 Mobile Phone Bezel Layout** directly from the UI. This enhances accessibility and gives citizens a real-world preview of how phishing scams render on their smart devices.

---

## 🚀 Key Features

*   **📚 Cyber Safety Learning Hub:** Core modules covering key defense strategies against everyday web threats.
*   **🔍 Scam Detection Simulators:** Interactive modules for Facebook, Instagram, Twitter/X, Snapchat, and WhatsApp phishing detection.
*   **🔑 Password Safety Validator:** Offline tool checking password entropy, strength rating, and key indicators.
*   **⚖️ Digital Rights Guide:** Quick access to the Electronic Transactions Act (ETA) of Nepal.
*   **🎓 Multi-Question Safety Quiz:** Interactive testing to verify digital literacy.
*   **🚨 Emergency Helpline (1144):** Direct connection to Nepal's Cyber Bureau resources.
*   **💻 Hybrid Terminal + GUI Interface:** Accessible to command line enthusiasts and everyday users alike.

## 🏃 Run Locally
Simply open `index.html` in any modern web browser or serve it locally using a light dev server:

```bash
# Using python simple server
python3 -m http.server 8000
```
Then visit `http://localhost:8000`.
