# Suraksha OS

Suraksha OS is an interactive citizen safety system designed to educate Nepali users about cybersecurity, digital rights, and safe digital practices.

Built using:
- HTML
- CSS
- Vanilla JavaScript

Features:
- Cyber awareness learning hub
- Scam detection simulator
- Password safety checker
- Digital rights guide (ETA Nepal)
- Privacy awareness tool
- Multi-question quiz
- Safety progress meter
- Emergency cyber help (1144)
- Terminal + Simple Mode UI

Goal:
Empower citizens to become digitally safer through guided interaction.
